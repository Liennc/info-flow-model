# Моделирование информационных потоков

## Описание
В данной работе рассматривается моделирование информационных потоков — процессов распространения информации во времени.

Были изучены и реализованы три математические модели:
- линейная
- экспоненциальная
- логистическая

## Данные
Для анализа использованы реальные данные из Google Trends по теме **ChatGPT**.

Данные отражают динамику популярности темы во времени.

## Метод
Для каждой модели были подобраны параметры с использованием метода наименьших квадратов (`curve_fit` из библиотеки SciPy).

Рассматривались следующие модели:
- Линейная: равномерный рост
- Экспоненциальная: ускоряющийся рост
- Логистическая: рост с последующим насыщением

## Метрика качества
Для оценки точности моделей использовалась среднеквадратичная ошибка (MSE).

## Результаты
В результате сравнения моделей было получено:
- Линейная модель — средняя точность
- Экспоненциальная модель — наименее точная
- Логистическая модель — наилучшее приближение

Логистическая модель показала наименьшую ошибку и лучше всего описывает реальные данные.

Вывод
Реальные информационные потоки имеют нелинейный характер.

Наиболее адекватной моделью является логистическая.
